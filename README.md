# it-s-here-logistics-dashboard
A functional logistics dashboard demo built for DelGate to showcase front-end capabilities, including charts, tables, filters, and responsive design.
This is a fully functional front-end demo of a logistics dashboard, built specifically as a showcase for DelGate. The app demonstrates a modern dashboard with deliveries, warehouse management, and analytics features.

Features
	•	Interactive dashboard cards showing delivery stats
	•	Bar, line, and pie charts for visualizing delivery and revenue data
	•	Search and filter functionality for delivery records
	•	Modal popups for detailed delivery information
	•	Dark/Light mode toggle
	•	Responsive layout for desktops and smaller screens

Note: This demo is built with plain HTML, CSS, and JavaScript for quick demonstration purposes. React and Tailwind are not used, but it is fully functional and ready for integration into a React + Tailwind setup.
